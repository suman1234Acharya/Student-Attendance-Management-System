# 🎓 Student Attendance Management System

A web-based Student Attendance Management System developed using PHP, MySQL, HTML, CSS, JavaScript, Bootstrap, and MySQL.

---

## 📌 Overview

This project helps educational institutions automate attendance management. It provides separate dashboards for administrators and class teachers to manage students, classes, attendance records, and reports.

---

## ✨ Features

- Admin Login
- Teacher Login
- Student Management
- Class Management
- Attendance Recording
- Attendance History
- Attendance Reports
- Dashboard
- Responsive Interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- XAMPP

## ⚙ Installation

1. Install XAMPP
2. Copy the project folder to:

```
htdocs/
```

3. Start Apache and MySQL.

4. Open phpMyAdmin.

5. Create a database.

```
attendance
```

6. Import the SQL file.

7. Open

```
http://localhost/Student-Attendance-Management-System
```
